class ipsuite.configuration_comparison.REMatch.REMatch(name: None)

Bases: ConfigurationComparison

REMatch kernel implementation.

Attributes

metric: str

The pairwise metric used for calculating the local similarity. Accepts any of the sklearn pairwise metric strings (e.g. “linear”, “rbf”, “laplacian”, “polynomial”) or a custom callable. A callable should accept two arguments and the keyword arguments passed to this object as kernel_params, and should return a floating point number.

alpha: float

Parameter controlling the entropic penalty. Values close to zero approach the best-match solution and values towards infinity approach the average kernel.

threshold: float

Convergence threshold used in the Sinkhorn-algorithm.
